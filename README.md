### 使用说明:
1. 登录cookie选项填入皮皮虾的cookie
2. 填入课件id(当前只支持填入一个课件id)
3. 填入存放pipixa配置的绝对路径
4. 填入要修改的config绝对路径
5. 点击开始修改岛配置
6. 批量修改题型时,用英文","隔开 (6674,6675)
